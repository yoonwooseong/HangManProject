## 🔨 기술 스택 및 구현 방식
 - Language : `Java`
 - Thread 사용 - BGM, IMAGE, GAMETIME(문제 푸는데 걸린 시간)   
 - StringBuffer & replace :플레이어 입력 및 문제맞춤 여부
 - File, Random - 힌트 및 난이도별 문제 가져오기
 - KeyAdapter = 입력 이벤트 버튼
